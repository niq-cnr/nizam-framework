# templates/

The `templates/` module owns the templates a consumer repository fills in after
bootstrap: `CONTEXT.md`, `AGENTS.md`, `DEBT.md`, an ADR template, and JSON/YAML
work-packet, phase, and manifest templates.

Full module content ships in feature 004 of phase 001-framework-genesis.
