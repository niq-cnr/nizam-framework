# schema/

The `schema/` module owns the JSON Schemas that validate every machine artifact the
framework and its consumers produce: frontmatter, manifest, phase, feature list,
contract, QA verdict, and run state.

Full module content (the seven canonical `*.schema.json` files) ships in feature 001 of
phase 001-framework-genesis.
