The `registry/` module owns the JSON Schema that the root `NIZAM.json` capability index
validates against, plus generalised scope-definition patterns.

Full module content (`nizam-index.schema.json`, `scope_definition_patterns.md`) and the
root `NIZAM.json` index ship in feature 006 of phase 001-framework-genesis.
